# Autoenv
Auto **activate** and **deactivate** if you use `virtualenv`

# Installation
`source /path/to/autoenv.sh` in .zhsrc or .bashrc

when you use `cd`
- **activate** env when into your project whatever your env name
- **deactivate** when you leave your project dir
